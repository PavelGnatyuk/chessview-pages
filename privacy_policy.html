üîó Third-Party Services

Chess.com API

When using the Chess.com integration feature, we connect to Chess.com's public API to retrieve your publicly available game data. This connection is made directly from your device and is subject to Chess.com's own privacy policy and terms of service.

OpenAI API

When using the AI analyzer feature, chess positions are sent to OpenAI's API for analysis. This data is processed according to OpenAI's Privacy Policy. We do not store or have access to this analysis data after it's displayed to you.
